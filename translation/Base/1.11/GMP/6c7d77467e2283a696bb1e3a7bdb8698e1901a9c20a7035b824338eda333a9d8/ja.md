```
BigInt(x)
```

任意精度の整数を作成します。`x` は `Int`（または `Int` に変換できるもの）である必要があります。この型には通常の数学演算子が定義されており、結果は [`BigInt`](@ref) に昇格されます。

インスタンスは、[`parse`](@ref) を介して文字列から構築することができるか、`big` 文字列リテラルを使用して構築できます。

# 例

```jldoctest
julia> parse(BigInt, "42")
42

julia> big"313"
313

julia> BigInt(10)^19
10000000000000000000
```
