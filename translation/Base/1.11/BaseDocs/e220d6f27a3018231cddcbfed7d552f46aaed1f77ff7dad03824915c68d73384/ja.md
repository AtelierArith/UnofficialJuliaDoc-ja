```
...
```

「スプラット」演算子 `...` は、引数のシーケンスを表します。`...` は関数定義で使用でき、関数が任意の数の引数を受け入れることを示します。`...` はまた、引数のシーケンスに関数を適用するためにも使用できます。

# 例

```jldoctest
julia> add(xs...) = reduce(+, xs)
add (generic function with 1 method)

julia> add(1, 2, 3, 4, 5)
15

julia> add([1, 2, 3]...)
6

julia> add(7, 1:100..., 1000:1100...)
111107
```
