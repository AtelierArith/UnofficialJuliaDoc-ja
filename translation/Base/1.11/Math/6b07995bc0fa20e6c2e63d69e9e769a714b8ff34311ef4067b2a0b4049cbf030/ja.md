```
@evalpoly(z, c...)
```

多項式 $\sum_k z^{k-1} c[k]$ を係数 `c[1]`, `c[2]`, ... に対して評価します。すなわち、係数は `z` の冪の昇順で与えられます。このマクロは、ホーナー法または、複素数 `z` の場合はより効率的なゴーツェル類似アルゴリズムを使用する効率的なインラインコードに展開されます。

詳細は [`evalpoly`](@ref) を参照してください。

# 例

```jldoctest
julia> @evalpoly(3, 1, 0, 1)
10

julia> @evalpoly(2, 1, 0, 1)
5

julia> @evalpoly(2, 1, 1, 1)
7
```
