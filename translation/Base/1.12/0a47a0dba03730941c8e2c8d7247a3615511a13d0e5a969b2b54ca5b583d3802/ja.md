```julia
===(x,y) -> Bool
≡(x,y) -> Bool
```

`x` と `y` が同一であるかどうかを判断します。これは、どのプログラムもそれらを区別できないという意味です。まず、`x` と `y` の型が比較されます。それらが同一であれば、可変オブジェクトはメモリ内のアドレスで比較され、不変オブジェクト（数値など）はビットレベルで内容が比較されます。この関数は時々「egal」と呼ばれます。常に `Bool` 値を返します。

# 例

```jldoctest
julia> a = [1, 2]; b = [1, 2];

julia> a == b
true

julia> a === b
false

julia> a === a
true
```
