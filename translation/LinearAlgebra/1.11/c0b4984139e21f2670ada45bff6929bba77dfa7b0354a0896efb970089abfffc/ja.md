```
muladd(A, y, z)
```

組み合わせた乗算加算、`A*y .+ z`、行列-行列または行列-ベクトルの乗算のためのものです。結果は常に `A*y` と同じサイズですが、`z` は小さいか、スカラーである可能性があります。

!!! compat "Julia 1.6"
    これらのメソッドはJulia 1.6以降が必要です。


# 例

```jldoctest
julia> A=[1.0 2.0; 3.0 4.0]; B=[1.0 1.0; 1.0 1.0]; z=[0, 100];

julia> muladd(A, B, z)
2×2 Matrix{Float64}:
   3.0    3.0
 107.0  107.0
```
