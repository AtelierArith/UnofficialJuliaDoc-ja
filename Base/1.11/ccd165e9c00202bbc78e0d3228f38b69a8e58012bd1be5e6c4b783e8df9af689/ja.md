```
<(x, y)
```

小なり比較演算子。[`isless`](@ref) にフォールバックします。浮動小数点の NaN 値の動作のため、この演算子は部分順序を実装します。

# 実装

標準的な部分順序を持つ新しい型は、新しい型の二つの引数に対してこの関数を実装する必要があります。標準的な全順序を持つ型は、代わりに [`isless`](@ref) を実装する必要があります。

関連情報として [`isunordered`](@ref) も参照してください。

# 例

```jldoctest
julia> 'a' < 'b'
true

julia> "abc" < "abd"
true

julia> 5 < 3
false
```
